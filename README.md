# Customer Behaviour Analysis Project

## Project Overview
This project simulates a professional end-to-end data analytics workflow, demonstrating the ability to turn raw data into actionable business insights.

### Key Components

- **Data Preparation, Modeling & Exploratory Analysis (Python):**  
  Clean, preprocess, and transform raw data for analysis.

- **Data Analysis (SQL):**  
  Simulate business transactions and run queries to uncover insights about customer segments, loyalty, and purchase behavior.

- **Visualization & Insights (Power BI):**  
  Build an interactive dashboard to highlight key trends and patterns, enabling stakeholders to make data-driven decisions.
